Aragubas-DiscordTheme

“That theme with a Compact design and a blue color scheme without performance compromises”

Simple and modern theme made with CSS and SCSS featuring:

  • Top Bar/Down bar layout
  • Hidden Server List
  • Blue Color Scheme
View on Github

Topbar/Downbar Layout

With its TopBar/DownBar layout its possible to make a compact layout with modern design very easily without compromising useability.

Hidden Server List

Its only visible when you need it, hover your mouse in the blank space on the left side to show the server list, its shows over the main content as a popup so it doesn’t change the size of the window, increasing performance since less re-paints will be needed.

The essential is invisible to the eye.
